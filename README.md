# Utilify

A project built with Angular 2. Utilizes the Spotify Api and Oauth2.0 for user verification. Verify the app with your account and view your current playlists. Select a specific playlist and search for suggested tracks based off of genre and a target tempo or BPM. Add tracks to your current playlist live. 

## Install

Clone this repo. 

cd into project directory. 

run npm install to resolve all dependencies. 

run ng serve to view on localhost/4200
